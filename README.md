# Explorer App Vue 3 Vite

## Getting Started
Install Dependencies (if counter error, try use node v22)
```bash
npm install
```

Copy Environment Variables and change VITE_API_URL value
```bash
cp .env.example .env
```

## Development
To start the development server run:
```bash
npm run dev
```
